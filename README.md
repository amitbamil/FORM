# University Registration Form

## Description

This is a simple University Registration Form created using HTML and CSS. It allows students to enter their personal, contact, academic, and account details.

## Features

* Personal Details Form

* Contact Information

* Course Selection

* Gender Selection

* Photo Upload

* Password Creation

* Security Question

* Terms and Conditions Checkbox

* Register and Reset Buttons

## Technologies Used

* HTML5

* CSS3

## Project Structure

University-Registration-Form/

│── project1.html

│── style.css

│── README.md

## How to Run

* Download or clone the repository.

* Open the project folder.

* Open project1.html in any web browser.
